# 📊 Employee Performance & Quality Dashboard

### 🎯 Objective
A comprehensive interactive dashboard built in Power BI to monitor employee performance, track fatal errors, and calculate overall quality scores across different supervisors and work locations. This project was developed as a Capstone for the **Workearly AI Data Analyst Bootcamp**.

### 🛠️ Key Features & Technical Skills

* **Advanced DAX Measures:** Utilized `DIVIDE` and custom logic to calculate `Sampling%`, `Defects%`, and dynamic `Quality Scores`.
* **Custom Tooltips:** Designed tailored report page tooltips to provide deep-dive analytics per employee upon hover.
* **Dynamic Visualizations:** Implemented Scatter charts with Play Axis to track fatal error trends over time, alongside mapped data for geographic location tracking.
* **Data Modeling:** Cleaned and structured raw quality data for optimal performance.
* **UI/UX Design:** Applied professional background themes and layouts to enhance readability and user engagement.
